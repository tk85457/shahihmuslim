## ✨ Features

- **Complete Hadith Collection:** Access the entire collection of Sahih Muslim hadiths seamlessly.
- **Beautiful UI/UX:** Clean, modern, and edge-to-edge typography and layout designed for readability.
- **Fast & Responsive:** High performance powered by Flutter, with native-like responsiveness.
- **State Management:** Robust architecture utilizing Riverpod for predictable state management.
- **Offline First:** Built on top of sqflite, ensuring the entire collection is available without an active internet connection.

## 🛠️ Technology Stack

- **Framework:** [Flutter](https://flutter.dev/) (Dart)
- **State Management:** [Riverpod](https://riverpod.dev/)
- **Routing:** [GoRouter](https://pub.dev/packages/go_router)
- **Database:** [sqflite](https://pub.dev/packages/sqflite)
- **Typography:** [Google Fonts](https://pub.dev/packages/google_fonts)

## 📦 Getting Started

### Prerequisites

- Flutter SDK (v3.24+ recommended)
- Dart SDK
- Android Studio / Xcode for deploying to devices.

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/tk85457/shahihmuslim.git
   cd shahihmuslim
   ```

2. **Fetch dependencies:**
   ```bash
   flutter pub get
   ```

3. **Run the app:**
   ```bash
   flutter run
   ```

## 🚀 Building for Release

### Android
To build an APK:
```bash
flutter build apk --release
```
To build an App Bundle (AAB):
```bash
flutter build appbundle
```
